Seating Chart

This project was a spin on normal seating charts, except every person in the class added themselves to their seats. When clicked, each person's picture would do some unique animation. The code for the entire class was hosted on a GitHub repository, however each person worked on their own local copy. We, as a class, learned how to merge our wildly different code together without conflicts, ensuring that we pooled our efforts into a working final seating chart.
